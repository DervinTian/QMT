# QMT #

## Motivation ##
Having taking classes in both database management systems and operating systems as a software engienering student, the purpose of this project was to test those skills and put the 
concepts learned from those classes, such as query optimization and filesystem management into use.

## Overview ##
The QMT Database is a filesystem-based database operating off of the QMT DDL. The QMT is a custom-made DDL that I implemented with a C++ backbone that operates very similarly to
commerical DDL's, such as SQL (albeit much simpler). 

### Codebase ###

The codebase portion of this repository holds all the code that brings this project to life. Notably it is mainly written in C++ version 20. 

Any AI usage is marked in the script with an AI comment above it. I try my best to write everything from scratch, but some things like pretty printing and documentation involved the use of AI to streamline the process and professionalize it.

### Documentation ### 

The documentation portion of this repository holds all of the documentation for this project. In the docs portion of the repository you will find the documentation
for each keyword that was implemented in the QMT DDL, along with usage examples and the implementation of the keyword command itself.

_This project was just for fun and to tie in concepts that I learned at University into one large project._
