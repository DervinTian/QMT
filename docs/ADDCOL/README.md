# ADDCOL # 

## Purpose ##
The purpose of the ADDCOL feature is to add a column to an existing database table.

## Usage ##
```cpp
ADDCOL (table_name, column_type, column_name);
```
- **table_name:** represents the name of the table that we want to add column (attribute) to.
- **column_type:** represents the type of the column (string, int, etc.)
- **column_name:** represents the name of the column to be added
