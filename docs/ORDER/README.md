# ORDER # 

## Purpose ##
The purpose of the ORDER keyword is to sort the queries based on the sorted values by a column's value.

## Usage ##
```cpp
ORDER (column_name, sort_order);
```
- **column_name:** represents the column name that we will be sorting on.
- **sort_order:** represents the direction that we want to sort. Available options are:
    - DESC: Sort by greatest value on top, with other values decreasing.
    - ASC: Sort by smallest value on top, with other values increasing.
