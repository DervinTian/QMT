# SET # 

## Purpose ##
The purpose of the SET keyword is to specify the values to set when using the UPDATE keyword function.

## Usage ##
```cpp
SET (column_name, column_value);
```
- **column_name:** represents the column that we want to update the value of.
- **column_value:** represents the new value that we set the previously specified column_name to.

## Example ##
```cpp
SET (name, "JOHN");
```

_Note: The table name is derived from the UPDATE statement above the SET statment(s)._
