# MOVE # 

## Purpose ##
The purpose of the MOVE keyword is to move the data from an original table over to a new table, leaving the original empty.

## Usage ##
```cpp
MOVE (source_table, dest_table);
...
```
- **source_name:** represents the name of the table that we want to source the data from.
- **dest_name:** represents the name of the table where we want to move the source data to.
